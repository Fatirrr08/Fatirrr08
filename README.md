### 🚀 Highlight Projects

#### 🛍️ [Salin Gaya - Premium Thrifting E-Commerce](https://github.com/Fatirrr08/salin-gaya-e-commerce)
Platform e-commerce thrifting premium yang responsif dengan integrasi AI.
* **Stack:** React, TypeScript, Vite, Tailwind CSS, Firebase, Gemini AI (`@google/generative-ai`).
* **Fitur:** Pencarian cerdas, antarmuka pengguna interaktif, dan fungsionalitas rekomendasi bertenaga AI.

#### 👁️ [El Presentasi (el_gestur_v2)](https://github.com/Fatirrr08/el_gestur_v2)
Sistem presentasi interaktif hands-free berbasis Computer Vision dan Voice Recognition.
* **Stack:** Python, YOLOv8, MediaPipe, Speech Recognition.
* **Fitur:** Mengontrol slide presentasi (PowerPoint, Canva, Figma) secara realtime menggunakan gestur tangan dan perintah suara.

#### 📈 [Perintis - UMKM Business Validator](https://github.com/Fatirrr08/Perintis)
Platform simulasi finansial dan validasi ide bisnis instan untuk UMKM di Indonesia.
* **Stack:** Python, HTML, CSS, JavaScript.
* **Fitur:** Kalkulasi kelayakan finansial kuantitatif dan analisis kualitatif berbasis AI.

#### 📚 [LibraryPro - Library Management System](https://github.com/Fatirrr08/Library-Pro)
Sistem informasi perpustakaan berbasis web dengan visual dashboard analitik.
* **Stack:** Java Web (JSP & Servlets), MVC Architecture, MySQL.
* **Fitur:** Dashboard analitik real-time, manajemen katalog buku, dan peminjaman otomatis.
